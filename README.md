# Mynd Focus Type Quiz (MVP)

Mobile-first, two-part quiz funnel aligned to Mynd brand cues.

## What it includes
- Part 1: 6-question identity quiz with result archetypes:
  - The Tab-Switcher
  - The Fried Founder
  - The Overthinker
  - The Creative Sprinter
- Part 2: Optional 4 timed puzzles (2 minutes each) with points
- Timeout behavior: if a puzzle times out, app recommends a product from Mynd
- Brand palette applied from live site CSS tokens (`#375555`, `#b5cfbc`)

## Run locally
Open [index.html](index.html) in your browser.

## File map
- `index.html` app shell
- `styles.css` mobile-first brand styling + motion
- `app.js` quiz state machine, scoring, puzzles, timer logic
